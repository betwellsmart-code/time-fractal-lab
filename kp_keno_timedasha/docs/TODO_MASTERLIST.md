## Mikro-Dasha Kalibrierung mit Währungsdaten (10s / 5s / 1s Charts)

**Status:** geplant  
**Priorität:** hoch  
**Bereich:** Zeitfraktale / Finanzanalyse / Mustererkennung  
**Zuständig:** Noreki + Agent 1 + Agent 5

### Ziel
Verwendung von hochfrequenten Finanzchart-Daten (Währungen, Indizes, Krypto) zur Untersuchung extrem schneller Dasha-Wechsel (unterhalb der Prana-Stufe).

### Warum FX?
- Milli-/Sekunden-genaue Zeitstempel  
- Keine Übertragungsverzögerung wie bei Darts  
- Reproduzierbare Daten  
- Ideale Testumgebung für fraktale Temporal-Modelle

### Aufgaben
- Datenfeed wählen (z. B. Binance, Oanda, MT5 Export)  
- 10s / 5s / 1s Zeitreihen importieren  
- Mikro-Dasha auf Sekundenbereich skalieren  
- Mustererkennung testen (PatternCore)  
- Wechselpunkte bestimmen (TemporalSynth)  
- Vergleich: FX vs KENO (Randomness-Test)

### Erwartete Ergebnisse
- Nachweis oder Widerlegung von Mikro-Dasha-Effekten  
- Robuste mathematische Grundlage  
- Neue Features für KI-Agenten (Temporal + Muster)
## Mikro-Dasha Analyse im Dartsport (experimentelle Stufe)

**Status:** geplant  
**Priorität:** niedrig-mittel  
**Bereich:** Zeitfraktale / Performance-Dynamik / Experimentalphysik  
**Zuständig:** Noreki + Agent 5 (TemporalSynth)

### Problemstellung
Darts-Livedaten enthalten Übertragungsverzögerungen (TV-Latency, Kameradrops).  
Es ist unklar, ob wir eine exakte Zeitachse rekonstruieren können.  
Projekt dient als *experimentelle Testumgebung* für Mikro-Dashas in Sportperformance.

### Herausforderungen
- Broadcast-Verzögerung variiert je nach Turnier  
- Unbekannte Sync-Offsets zwischen Scoreboard und Wurf  
- Nicht jedes Event bietet exakte Millisekunden-Zeitstempel  

### Aufgaben
- Untersuchung möglicher Datenquellen mit präziser Zeitmarkierung  
- Analyse von Timing-Abweichungen  
- Entwicklung einer Korrekturkurve (Kalibrier-Funktion für Latenz)  
- Vergleich: Mikro-Dasha-Effekte im Darts vs. Währungscharts  
- Integration in PatternCore: Reaktion auf „Druckpunkte“  
- Integration in TemporalSynth: Mikro-Energiewechsel im Wurfverlauf  

### Erwartetes Ergebnis
- Hypothesenreduktion: Was ist echte Leistung, was ist Timing?  
- Erkenntnisse über Mikro-Zeitdynamik im Sport  
- Vorbereitung für spätere Live-Modelle (wenn bessere Daten verfügbar sind)
